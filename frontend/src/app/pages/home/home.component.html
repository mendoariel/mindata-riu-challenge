Requisitos Técnicos

1 Framework

  • ✓ Angular 21+ con componentes standalone
  • ✓ Todos los componentes son standalone

2 Styling

  • ✓ Tailwind CSS 4 instalado
  • ✓ @tailwindcss/postcss configurado

3 Responsive Design

  • ✓ Adaptación a breakpoints del diseño (mobile, tablet, desktop)
  • ✓ Adaptación fluida con Tailwind CSS

4 Accesibilidad

  • ✓ HTML5 semántico (header, main, section, article, etc.)
  • ✓ Roles ARIA para iconos decorativos

5 Imágenes

  • ✓ Implementación fluida con object-fit: cover
  • ✓ Placeholders con proporciones correctas

Criterios de Evaluación

NO se evaluará

  • • Lógica de negocio
  • • Manejo de estados
  • • Funcionalidad interactiva (solo layout)

SÍ se evaluará

  • Fidelidad al diseño de Figma
  • Responsive design fluido
  • Uso correcto de Tailwind CSS 4
  • Estructura de componentes standalone
  • Accesibilidad (HTML5 semántico y ARIA)
  • Implementación de imágenes responsive

Nota Importante

Este proyecto se enfoca exclusivamente en la maquetación del layout. No se implementa lógica de negocio ni manejo de estados, ya que el challenge evalúa únicamente la fidelidad al diseño, el responsive design, el uso de Tailwind CSS 4, la estructura de componentes y la accesibilidad.
